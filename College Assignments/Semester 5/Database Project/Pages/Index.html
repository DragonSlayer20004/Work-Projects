Study Pages

We have three wonderful pages full of information for you to study!

We have pages each dedicated to the three main types of rocks!

Sedimentary Rocks which are formed by the accumulation of sediments.

Igneous Rocks which are fromed from the solidification of molten rock.

Metamorphic Rocks which have been modified from intense heat and pressure from beneath the surface.

Obsidian is a naturally occurring volcanic glass formed when lava extruded from a volcano cools rapidly with minimal crystal growth.

Memory Game

Memory games will test your skill at remembering information and its connecting rocks.

With several modes you choose which one best fits your skill level!

Simple matching game is a easy memory game where you pair rocks with each other.

Naming matching game takes the difficulty up a notch as you need to compare rocks with their associating names.

Descriptive matching game is the hardest as you have to guess the rock of the description given.

Guessing Game

We have a very fun guessing game to help you learn about rocks.

Each game is randomized to keep you on your toes! There are even unique modes to play!

Image guess game shows you a rock and you have to guess what it is!

Description guess game gives you a short description and you have to guess what rock it is!
